1. Распакуйте архив в удобном для вас месте (в папку)
2. Установите python последней версии https://www.python.org/downloads/
3. Откройте командную строку (пропишите cmd в пуск)
4. Перейдите в директорию где распокавали файлы и скопируйте путь
5. Пропишите в командой строке: cd *путь который вы скопировали*
6. Выполните команду: pip install -r requirements.txt дождитесь окончания загрузки
7. Откройте файл config.py через блокнот, ии вставьте токен бота (его можно получить тут - https://t.me/BotFather)
8. Запустите файл main.py с помощью python (или выполните пункт 4 и 5, и пропишите в командной строке: python main.py)
9. Напишите /start боту для начала работы
